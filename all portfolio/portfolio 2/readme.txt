# News Scraper Bot  

## **Description**  
The News Scraper Bot is a simple web scraping tool that fetches the latest news headlines from BBC News. It displays the top 5 news articles in the terminal and allows users to open selected articles in their web browser.  

## **Features**  
✔ Scrapes the latest news headlines using `requests` and `BeautifulSoup`  
✔ Displays the top 5 news articles in a user-friendly format  
✔ Allows users to open articles in their default web browser  

## **Installation**  
Ensure you have Python installed, then install the required libraries:
