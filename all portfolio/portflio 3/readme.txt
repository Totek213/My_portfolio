# Task Manager Bot  

## **Description**  
The Task Manager Bot is a simple command-line to-do list application that allows users to manage their tasks efficiently. Tasks are stored in a JSON file, ensuring they persist between sessions. Users can add new tasks, mark them as completed, delete tasks, and view their task list.  

## **Features**  
✔ Add new tasks to the to-do list  
✔ List tasks with a ✔ (completed) or ✖ (pending) status  
✔ Mark tasks as completed  
✔ Delete tasks from the list  
✔ Persistent storage using a JSON file  

## **Installation**  
No additional libraries are required. The bot runs with standard Python libraries.
