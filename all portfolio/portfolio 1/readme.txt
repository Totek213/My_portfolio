# Voice Assistant Bot  

## **Description**  
The Voice Assistant Bot is an AI-powered virtual assistant that listens to voice commands, processes them using OpenAI's GPT model, and responds with both text and speech. It enables hands-free interaction and can answer questions, provide information, and assist with various tasks.  

## **Features**  
✔ Converts speech to text using `speech_recognition`  
✔ Processes user input with OpenAI's API  
✔ Responds with synthesized speech using `pyttsx3`  
✔ Hands-free interaction for convenience  

## **Installation**  
Make sure you have Python installed, then install the required libraries:  
```bash
pip install openai speechrecognition pyttsx3 pyaudio
